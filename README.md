\# PROJECT CONTEXT: 2-Stage Ballistic Missile GNC Simulator



\## 1. Project Goal

Development of a 2DOF (Degree of Freedom) closed-loop GNC (Guidance, Navigation, and Control) simulator for a 2-stage ballistic missile (IRBM class). The objective is to maximize the final ballistic downrange using numerical optimization (`fmincon`) while respecting structural constraints (aerodynamic load limits < 60 kN), implementing the physical plant and the control loop in Simulink.



\## 2. Technical Architecture

The software architecture is divided into three interconnected layers:

\* \*\*Layer 1: Optimization \& Initialization (MATLAB Scripts)\*\*

&#x20;   \* Use of `fmincon` to solve the boundary value problem.

&#x20;   \* Decision variables: Parameters for the Bilinear Tangent steering law and Gravity Turn initiation.

&#x20;   \* Cost function: Maximize downrange ($\\psi$).

\* \*\*Layer 2: Plant / Vehicle Dynamics (Simulink)\*\*

&#x20;   \* \*\*Translational Dynamics:\*\* 2DOF point-mass equations over a spherical, rotating Earth. Includes variable gravity ($g(z)$), ISA atmosphere ($\\rho(z)$), and aerodynamic drag ($D$).

&#x20;   \* \*\*Rotational Dynamics:\*\* Rigid body pitch dynamics. Calculates pitch moment ($M$) generated by Thrust Vector Control (TVC), resulting pitch rate ($q$), attitude angle ($\\theta$), and flight path angle ($\\gamma$).

&#x20;   \* \*\*Mass Management:\*\* Continuous mass flow rate ($\\dot{m}$) calculation and discrete staging events (instantaneous structural mass drop between Stage 1 and Stage 2).

\* \*\*Layer 3: GNC Loop (Simulink)\*\*

&#x20;   \* \*\*Guidance:\*\* Reference signal generator based on Gravity Turn (Stage 1) and Bilinear Tangent Law (Stage 2). Outputs desired flight path angle ($\\gamma\_{ref}$).

&#x20;   \* \*\*Control:\*\* PID controller governing the TVC actuator. Calculates required nozzle deflection ($\\delta\_c$) to track $\\gamma\_{ref}$.
